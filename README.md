## 프로젝트 소개

오프라인 마피아 게임을 위한 진행 가이드

## 기술 스택

### Front-end

Next.js, TypeScript, Tailwind CSS, XState, Recoil, React Spring, PWA

### Environment

Vercel, VS-code, Git, GitHub

## 이용 안내

- 마스터 제외, 5~10 명 플레이어 수 제한
- 6명 이상 경찰 직업 가능
- 7명 이상 의사 직업 가능
